Cara Implementasi di Project Anda:

  1. Upload model 3D Anda (format .glb atau .gltf) ke server
  2. Copy kode model-viewer dan ganti URL model dengan model Anda

  4. Pastikan website menggunakan HTTPS (wajib untuk AR)
  5. Test di berbagai device untuk memastikan compatibility
